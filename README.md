## Privacy Policy

**Last updated:** March 2026

### 1. Overview
This Privacy Policy describes how sns-autoposter handles information when you use our social media automation service.

### 2. Information We Access
- **Social Media Credentials:** OAuth access tokens for TikTok, YouTube, Instagram, and LINE VOOM (stored locally, never shared)
- **Content Data:** Video files and metadata that users provide for posting
- **Google Drive & Sheets:** Files and spreadsheet data that users explicitly authorize

### 3. How We Use Information
- To authenticate with social media platforms on behalf of the user
- To upload and publish content as directed by the user
- To read scheduling and metadata from user-controlled spreadsheets
- To generate captions using AI (Anthropic Claude API) when requested

### 4. Data Storage
- All credentials and tokens are stored locally on the user's device
- No user data is stored on external servers operated by this App
- Temporary video files are deleted immediately after posting
- No personal data is sold or shared with third parties

### 5. Third-Party Services
This App integrates with: TikTok API, YouTube Data API, Instagram Graph API, LINE VOOM API, Google Drive API, Google Sheets API, Anthropic Claude API.

### 6. Data Security
API keys and access tokens are stored in environment variables and never committed to version control or shared publicly.
